## Technical Integration Notes

### External Links
All links should:
- Open in new tab (`target="_blank"`)
- Include `rel="noopener noreferrer"`
- Track clicks for analytics

### Potential Embeds
- SprayFoamTV: Video iframe or preview
- SprayFoamRadio: Audio player widget
- NerfFusion: Product cards (if API available)

### Fallbacks
- Static cards with screenshots if embeds unavailable
- Descriptive text for accessibility
- Loading states for dynamic content
