## Navigation Menu
| Menu Item | URL | Description |
|-----------|-----|-------------|
| Home | `/` | Landing page with all sections |
| Shop | `/shop/` | Product catalog |
| Cart | `/cart/` | Shopping cart |
| Checkout | `/checkout/` | Payment and order completion |
| My Account | `/my-account/` | User dashboard, login, registration |
| Login/Signup | `/my-account/` | Authentication |

## Page Types

### Product Pages
- Product image gallery
- Title and price range
- Size selector (S, M, L, XL, 2XL, 3XL)
- Color/material selector
- Detailed description
- Add to cart button
- Related products section
- Reviews section

### Account Pages
- Login form
- Registration form
- Password recovery
- Order history
- Account details

### Cart Page
- Product list with quantities
- Price breakdown
- Coupon code field
- Checkout button
- "Return to shop" link (when empty)

### Checkout Page
- Billing information form
- Shipping information
