# Foam Everything - Site Overview

## Website Information
- **URL:** https://foameverything.com
- **Platform:** WordPress with Divi Theme
- **E-commerce:** WooCommerce
- **Industry:** Spray Foam Insulation Apparel & Merchandise

## Purpose
Foam Everything is an e-commerce website selling branded merchandise for the spray foam insulation industry. The site features:
- Custom apparel (t-shirts, hoodies, longsleeves)
- NASCAR racing-themed merchandise
- 3D brand character design services
- UV coated decals
- Virtual metaverse experiences (Spatial.io integration)

## Target Audience
- Spray foam insulation professionals
- Industry enthusiasts
- NASCAR/racing fans
- Companies wanting custom branded merchandise
